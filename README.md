# Ankit Portfolio (Next.js)

Professional one-page portfolio for job search.

## Run locally

```bash
npm install
npm run dev
```

Open: http://localhost:3000

## Build

```bash
npm run build
```

Static output is generated in `out/`.

## Deploy

Push to `main` and GitHub Actions deploys automatically to GitHub Pages.

Expected URL:

`https://vermaankit2005.github.io/ankit-portfolio-next/`
